# CTB评委现场打分系统 Demo

一个面向全国站现场评审场景的产品 Demo，用于展示评委评分、工作人员巡检、异常识别、结果汇总与导出的完整闭环。

## 项目简介

本 Demo 聚焦一个高并发、多角色、强时效的线下评审场景：20 个房间并行评审、每房 8 位评委、共 1000 支队伍分批展示。产品目标不是单纯做一个评分表，而是让现场评分流程更稳定、更可控，并让工作人员能实时发现异常、快速推进闭环。

## 核心价值

- 让评委在最短路径内完成当前队伍评分
- 让工作人员一屏掌握房间进度与异常状态
- 让未收齐评分的队伍被及时识别，而不是等到汇总阶段才暴露问题
- 让已完成队伍自动进入结果汇总，并支持导出

## Demo包含的功能

- 指挥总览：查看全场评分态势、风险事件和可汇总队伍
- 评委工作台：对当前队伍进行四维度评分、校验并提交
- 房间监控：查看单房间进度、已提交/未提交评委和异常队伍
- 成绩中心：展示实时排名、房间出分概况和待处理队伍
- 结果导出：支持导出 CSV 结果快照

## 适用角色

- 评委：快速完成独立评分
- 工作人员：实时查看房间进度、定位异常并跟进处理
- 管理员 / 总控：查看全局完成情况并导出结果
